# 🛠️ Config Server

Este servicio actúa como un **Config Server centralizado**, proporcionando configuraciones a otras aplicaciones dentro del ecosistema. Las configuraciones se almacenan en un repositorio Git.

---

## 📦 Características

- Gestión centralizada de configuraciones para múltiples entornos.
- Backend basado en Git (GitHub).
- Integración con Azure Key Vault para manejar secretos de forma segura
- Compatible con despliegue en Docker y Kubernetes.
